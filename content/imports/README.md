# Iqbal Academy import queue

Files in this directory are source transcriptions imported from Iqbal Academy.
The first seven *Bal-e-Jibril* poems were visually compared with printed pages
21-27 (PDF pages 22-28) of the local scan on 2026-06-19. The file records the
exact PDF pages checked for each poem.

Run `npm run import:iqbal-academy` to rebuild the first seven *Bal-e-Jibril*
imports. The script normalizes Unicode encoding variants only; it does not ask
Gemini or OpenAI to rewrite or correct Iqbal's verse. Verified imports are
accepted by the Gemini draft validator but are not automatically published.
